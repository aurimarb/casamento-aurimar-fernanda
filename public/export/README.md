# Imagens do site (export)

Coloque aqui as imagens PNG exportadas junto com os HTML da pasta `EXPORT`.

**Arquivos esperados (nomes do layout Web_1920):**

- `aac.png` — Banner do topo (hero)
- `aak.png` — Foto da seção “Boas-vindas”
- `aah.png` — Foto da seção “Nossa história”
- `aba.png` — Ícone “Como chegar”
- `abb.png` — Ícone “Confirme presença”
- `abc.png` — Ícone “Lista de presentes”
- `abd.png` — Ícone “Madrinhas e Padrinhos”
- `acb.png` — Imagem do rodapé

Se as imagens estiverem na pasta `EXPORT/` na raiz do projeto, execute:

```bash
npm run copy-images
```

Isso copia todos os `.png` de `EXPORT/` para `public/export/`.
